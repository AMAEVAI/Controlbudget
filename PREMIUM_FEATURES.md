# SipSpend Premium Features

## Included in this update

- **English + French** via `Localizable.xcstrings` (iPhone language: Settings → General → Language)
- **Onboarding** (3 screens)
- **Brand logo** on Home
- **Dark / Light / System** theme (Settings)
- **Face ID lock** (Settings)
- **Quick amounts** on Add (€1, €2.50, €5, €10)
- **Repeat last transaction**
- **Search** in Activity
- **Edit transaction** (tap row)
- **Export CSV** (Settings)
- **Weekly drink goals** + progress on Home
- **Evening reminder** (8 PM)
- **Siri Shortcut** “Log drink” (App Intents)
- Budget alerts **80% / 100%** (localized notifications)

## French

Set iPhone to **Français** → restart app → UI and notifications use French strings.

## Siri

Say: *“Log drink in SipSpend”* → opens app and logs last drink amount.

## Widget

See `WIDGET_AND_NOTIFICATIONS.md` for widget target setup in Xcode.

## App icon

Replace images in `Assets.xcassets/AppIcon.appiconset` with your final 1024×1024 icon (use brand teal `#2E85D1`).
